What This Site Is

Cattaneo Grid (cattaneogrid.kevinroberts.ar) is a fan-made, non-commercial web application that provides a sortable grid and player interface for Hernán Cattaneo's Resident podcast. No audio content is stored or re-hosted on this site — all audio streams directly from PodBean's CDN (mcdn.podbean.com) via publicly available URLs. Episode metadata (titles, dates, download counts, tracklists) is aggregated from publicly accessible podcast feeds.
